## APIs
- **Affirm Checkout API** — Initialize and authorize Affirm checkout flows for online merchants.
- **Affirm Charges API** — Capture, void, and refund Affirm charges; read transaction details.
- **Affirm Transactions API** — Read transactions, settlements, and disputes.
